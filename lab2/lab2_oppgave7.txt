1. Hvilke ord gir opphav til forkortelsen CSS?
Cascading style sheet.

2. En CSS-regel angis skjematisk som følger, Hva heter de ulike komponentene (A, B og C)?:
A {
B : C;
}


A = element / declaration
B = Property
C = Value

3. Hvilke tre kilder (origin) gir opphav til CSS-regler når vi laster inn en vevside?
external, internal and inline style sheets. 

4. Dersom to ulike CSS-regler fra samme kilde har samme spesifisitet, men gir motstridende presentasjonsinformasjon. Hvilken av reglene gjelder da?
Sist skrevet, den overskrider den som ble skrevet først.
